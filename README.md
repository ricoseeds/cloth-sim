# Cloth-simulation

Cloth-Simulation using mass-spring system and opengl-3.3 in c++11.

### Run
```
    $ g++ src/*.cpp -I/usr/local/Cellar/eigen/3.3.7/include/eigen3 -framework OpenGl -I/usr/local/include -lglfw -lGLEW
```

## Built With
* Opengl3.3(with shaders)
* c++11
