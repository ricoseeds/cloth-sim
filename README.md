# Cloth-simulation

Cloth-Simulation using mass-spring system.

### Run
```
    $ g++ src/*.cpp -std=c++11 -I/usr/local/Cellar/eigen/3.3.7/include/eigen3 -framework OpenGl -I/usr/local/include -lglfw -lGLEW
```

## Built With
* OpenGl-3.3
* c++11
